C++ HELLO WORLD: overview prerequisites details

Overview:

Ahhh, the "Hello World" program. The simplest program that can possibly be written in any language, and a classic test to verify that your system is configured properly and all the necessary tools (compilers/interpreters, linkers, etc) are installed an working properly.

Prerequisites:

This tutorial is written assuming you have a functional Linux OS , and have installed the essential build tools. E.g. on ubuntu:

$ sudo apt-get install build-essential

You should also have a working knowledge of the command terminal.

See:

  • Installing Linux
  • Intro to BASH
