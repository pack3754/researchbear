Overview:

Taking it upon yourself to learn how to use the Git system can be an arduous task - particularly if you are collaborating with several people.

This tutorial is not (by any measure) comprehensive, but it contains some useful tips and rerences that should help make your Git experience a happy one (i.e. an experience with minimal crying and fetal-positioning...results may vary)

Figure 1
Git Cat-opus (?) is here to help! But cross him and he will destroy all you love!

Prerequisites:

Get Git! Hop over to GitHub to download the GitHub app. Windows and Mac have a GUI-style application that makes things a bit easier in a way, but keep in mind that the Linux version of Git will excersize your BASH muscles and get you all the ladies/gents (but seriously, the terminal is your friend)!
