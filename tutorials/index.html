Tutorials -

Here you will find an extensive set of tutorials on everything ranging from project management, to software development and electrical/mechanical design. The aim is to provide you with all the skills you need to perform outstanding research.

Project Management

So you want to be a physicist... Well, no matter what you want to be when you grow up, good project management skills are important to your success. The project management tutorials are designed to help you learn the tools and skills that will help you build a truly great project. This means communication, collaboration, documentation, presentation, and more!

  • Intro to Project Management - UNDER CONSTRUCTION - A broad overview of good project management practices, including advice and tips for getting started with your own project.
  • Project Management Guidelines - UNDER CONSTRUCTION - A specific set of guidelines to help you build a project that conforms with the standards that are encouraged on this site.
  • Advanced Project Management - COMING SOON - Tips for expanding your project to include things like build systems (cmake), documentation frameworks (doxygen), testing frameworks (gtest), and package distribution utilities (dpkg).

Software

Whether you are planning to do pure theoretical physics, build a flying robot, or analyze data from ultra high-speed cameras, chances are you will need to develop some software to support your project. This could mean developing a computer simulation, writing firmware to interface with a motor controller, or performing real-time image analysis. The software tutorials are designed to help you learn tools and practices that will aid in the design and implementation of software.

  • History of GNU/Linux - UNDER CONSTRUCTION - A (very) brief history of GNU/Linux.
  • Installing Linux - COMING SOON! - Learn how to install and configure our favorite linux distribution (Ubuntu).
  • Intro to BASH - COMING SOON! - Learn the basics of the popular BASH command shell.
  • C++ Hello World - COMING SOON! - learn how to write and compile your very own Hello World program in C/C++.

Hardware/Mechanical

Many physics projects will require mechanical design work. The hardware and mechanical design tutorials will help you learn the tools you need to create modern designs using Computer Aided Design (CAD) software (see Resources for a list of recommended CAD programs).

  • Intro to AutoDesk Inventor '11

Electrical

Electrical design is an important aspect of nearly any modern physics project. The electrical design tutorials will help you learn the tools and design principles required to create working electronic devices, embedded circuit boards, etc..
