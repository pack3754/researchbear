Overview

This site provides UNCo physics students with a comprehensive set of resources to aid in their undergraduate research projects. The site covers topics ranging from hardware and software design practices, to establishing good project management guidelines, and even where to find the best web comics (e.g. xkcd, SMBC, and Abstruse Goose).

Once you've spent a few hours catching up with those web comics, you can move on to the rest of the site. Enjoy!

Quick Start Guide:

  • Tutorials - If you are new to UNCo Physics, new to this page, new to GitHub and version control, or just want to browse some nifty tutorials, then this is the page for you.
  • Research - The main focus of this site - learn about ongoing and past research projects, and how you can get involved!
  • Demos - Coming soon! Past Physics students have teamed up with Dr. Willis to compile a set of awesome online physics demos.
  • Resources - Get connected with the official physics department pages, and 3rd party resources.

Additional Notes:

As you browse through this site, you will find the authors are _strong_ proponents of using a version control system to keep all of their project files organized, backed up, and synchronized. Of course, we have used one of our favorite version control systems (github) to work on this very page. To view the github repository for this page's source (HTML and CSS): Github Repo
